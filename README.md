# 숲 버츄얼 키우기 (SOOP Virtual Manager)

가상의 스트리밍 방송 환경에서  
**분탕 채팅을 관리하며 스트리머의 멘탈을 지키는 실시간 매니저 시뮬레이션 게임**

## 🎮 게임 소개

**숲 버츄얼 키우기**는  
빠르게 흘러가는 채팅 속에서 분탕을 골라내고  
올바른 제재(채금 / 강퇴 / 블랙)를 선택해  
스트리머의 멘탈을 유지하며 방송을 성장시키는 게임이다
